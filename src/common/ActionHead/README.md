# FormHead

A UI element for displaying a heading for a Pay Theory form.

## Hooks

-   None

## Props

### Required

-   header: the text to display as the header
-   action: function that will run on button click
-   label: label for the button that is in the ehader
-   actionId: id and data-testid that will be passed to the button

### Optional

-   icon: string that lets the module know if you want an icon on the button
